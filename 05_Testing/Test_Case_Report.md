# Test Case Report – Vocabulary Quiz Generator

ระบบ: Vocabulary Quiz Generator (Python Console Application)  
วัตถุประสงค์: ทดสอบการทำงานของฟังก์ชันหลักทั้งหมด รวมถึงกรณี Success, Error และ Boundary case

---

## สรุปภาพรวมการทดสอบ

- Test Cases ทั้งหมด: 12 เคส
- ฟังก์ชันที่ครอบคลุม:
  - เมนูหลัก (Main Menu)
  - การโหลดไฟล์คำศัพท์ (Load vocabulary file)
  - การเริ่มทำแบบทดสอบ (Start Quiz)
  - การตรวจคำตอบ (Correct / Incorrect)
  - การจัดการ Input ผิดรูปแบบ
  - การแสดงผลสรุปคะแนน (Quiz Summary)

---

## รายละเอียด Test Cases

> หมายเหตุ: คอลัมน์ **Actual Result** และ **Status** ให้ผู้ทดสอบกรอกหลังจากรันทดสอบจริง

| TC ID | หัวข้อทดสอบ | เงื่อนไขก่อนทดสอบ (Precondition) | ขั้นตอนการทดสอบ (Steps) | ข้อมูลนำเข้า (Input) | ผลลัพธ์ที่คาดหวัง (Expected Result) | Actual Result | Status |
|------|--------------|------------------------------------|--------------------------|------------------------|----------------------------------------|---------------|--------|
| TC01 | แสดงเมนูหลักถูกต้อง | เปิดโปรแกรมสำเร็จ (`python3 main.py`) | 1. รันโปรแกรม<br>2. สังเกตหน้าจอแรก | - | แสดงเมนูหลักพร้อมตัวเลือก `1. Load vocabulary file`, `2. Start quiz`, `3. Exit` ครบถ้วน | | |
| TC02 | เลือกเมนูที่ไม่ถูกต้อง | อยู่ที่หน้าจอเมนูหลัก | 1. รันโปรแกรม<br>2. กรอกค่า `9` ที่เมนูหลัก | `9` | ระบบแสดงข้อความแจ้งเตือนว่าเมนูไม่ถูกต้อง เช่น `Invalid menu option. Please select 1-3.` และแสดงเมนูหลักอีกครั้ง | | |
| TC03 | โหลดไฟล์คำศัพท์สำเร็จ (ใช้ไฟล์ vocab.csv) | มีไฟล์ `vocab.csv` อยู่ในโฟลเดอร์เดียวกับ `main.py` | 1. เลือกเมนู `1` (Load vocabulary file)<br>2. เมื่อถามชื่อไฟล์ กด Enter เปล่าเพื่อใช้ default (vocab.csv) | Enter เปล่า | ระบบแสดงข้อความ `[INFO] Loaded X questions from 'vocab.csv'.` (X = จำนวนคำถาม) และไม่ขึ้น Error | | |
| TC04 | โหลดไฟล์ไม่สำเร็จ (ไฟล์ไม่พบ) | ไม่มีไฟล์ชื่อ `notfound.csv` ในโฟลเดอร์ | 1. เลือกเมนู `1`<br>2. กรอกชื่อไฟล์ `notfound.csv` | `notfound.csv` | ระบบแสดง `[ERROR] File 'notfound.csv' not found.` และไม่ Crash โปรแกรมยังกลับไปเมนูหลักได้ | | |
| TC05 | เริ่มทำแบบทดสอบโดยยังไม่โหลดไฟล์ | เปิดโปรแกรมใหม่ และยังไม่ได้กดเมนู 1 | 1. เลือกเมนู `2` (Start quiz) ทันที | `2` | ระบบแสดง `[WARNING] No questions loaded. Please load vocabulary file first.` และไม่เข้าโหมดถามคำถาม | | |
| TC06 | เริ่มทำแบบทดสอบหลังจากโหลดไฟล์แล้ว | โหลดไฟล์ `vocab.csv` สำเร็จแล้ว (จาก TC03) | 1. เลือกเมนู `2` (Start quiz)<br>2. สังเกตว่ามีคำถามขึ้น | `2` | ระบบแสดงข้อความ `=== Start Vocabulary Quiz ===` และแสดงคำถาม `Question 1/X` พร้อมตัวเลือก 1–4 | | |
| TC07 | ตอบคำถามถูกต้อง | อยู่ในระหว่างการทำ Quiz และรู้คำตอบที่ถูกของคำถามนั้น | 1. ดูคำถามและตัวเลือก<br>2. กรอกหมายเลขตัวเลือกที่ถูกต้อง | เช่น `1` (ถ้าตัวเลือก 1 คือคำตอบถูก) | ระบบแสดง `✅ Correct!` และเมื่อจบทุกข้อ จำนวนคะแนน (Score) เพิ่มขึ้นตามจำนวนข้อที่ตอบถูก | | |
| TC08 | ตอบคำถามผิด | อยู่ในระหว่างการทำ Quiz | 1. ดูคำถามและตัวเลือก<br>2. เจตนากรอกคำตอบที่ผิด | ตัวเลข 1–4 ที่ไม่ใช่คำตอบถูก | ระบบแสดง `❌ Incorrect.` และคำถามนั้นถูกเก็บในรายการ `incorrect_questions` เพื่อนำไปแสดงใน Summary | | |
| TC09 | กรอกคำตอบที่ไม่ใช่ตัวเลข | อยู่ในระหว่างการทำ Quiz ที่แสดงตัวเลือก 1–4 แล้ว | 1. เมื่อถาม `Your answer (1-4)`<br>2. กรอกข้อมูลเป็นตัวอักษร เช่น `abc` | `abc` | ระบบแสดงข้อความ `Invalid input. Please enter a number.` และถามซ้ำ จนกว่าจะใส่ตัวเลขถูกต้อง โดยไม่ Crash | | |
| TC10 | กรอกคำตอบตัวเลขนอกช่วง (น้อยกว่า 1 หรือมากกว่า 4) | อยู่ในระหว่างการทำ Quiz | 1. เมื่อถาม `Your answer (1-4)`<br>2. กรอกค่า `0` หรือ `5` | `0` หรือ `5` | ระบบแสดงข้อความ `Please enter a number between 1 and 4.` และถามซ้ำ จนกว่าจะกรอกเลขในช่วง 1–4 | | |
| TC11 | แสดงสรุปผลหลังทำแบบทดสอบครบทุกข้อ | ทำ Quiz จนหมดทุกคำถาม | 1. ตอบคำถามทุกข้อจนจบ<br>2. สังเกตหน้าจอสรุป | ตามลำดับคำถาม | ระบบแสดงสรุปผล เช่น `Score: A/B (XX.XX%)` พร้อมจำนวน `Correct`, `Incorrect` และถ้ามีข้อผิด แสดงรายการคำศัพท์ที่ควรทบทวน | | |
| TC12 | ออกจากโปรแกรมจากเมนูหลัก | อยู่ที่หน้าจอเมนูหลัก | 1. เลือกเมนู `3` (Exit)<br>2. สังเกตการทำงานของโปรแกรม | `3` | ระบบแสดงข้อความ `Goodbye!` และปิดโปรแกรม กลับสู่ Terminal ปกติ | | |
