# คู่มือผู้ใช้งานระบบ Vocabulary Quiz Generator (ภาษาไทย)

ระบบ Vocabulary Quiz Generator เป็นโปรแกรมแบบ Console สำหรับฝึกคำศัพท์ภาษาอังกฤษ  
โดยอ่านคำศัพท์จากไฟล์ CSV และแสดงคำถามแบบตัวเลือก **5 ตัวเลือก** พร้อมสุ่มตำแหน่งคำตอบใหม่ทุกครั้ง  

---

# 1. ความต้องการระบบ (System Requirements)
- macOS / Windows / Linux
- Python 3.11+
- ไฟล์ vocab.csv อยู่ในโฟลเดอร์เดียวกับ main.py

---

# 2. การเริ่มใช้งานโปรแกรม
เปิด Terminal แล้วพิมพ์:

cd 04_Implementation/src

รันโปรแกรมด้วย:

python3 main.py

---

# 3. เมนูหลักของระบบ

1. Load vocabulary file  
2. Start quiz  
3. Exit  

---

## 3.1 Load vocabulary file
- โหลดไฟล์ vocab.csv หรือไฟล์ชื่ออื่นที่ระบุ  
- ถ้าโหลดสำเร็จ → แจ้งจำนวนคำศัพท์  
- ถ้าไม่พบไฟล์ → แจ้ง Error  

---

## 3.2 Start quiz
ใช้เริ่มทำแบบทดสอบคำศัพท์  
จะต้องโหลดไฟล์ก่อน

ระบบแสดง:
- คำศัพท์
- ตัวเลือก 5 ตัวเลือก (สุ่มลำดับ)
- ช่องสำหรับใส่คำตอบ 1–5

ระบบตรวจสอบ:
- ตอบถูก → Correct!
- ตอบผิด → Incorrect.
- ใส่ตัวอักษร → แจ้งเตือน
- เลขนอกช่วง → แจ้งเตือน

---

## 3.3 Exit
ออกจากโปรแกรม

---

# 4. ระหว่างทำแบบทดสอบ
แต่ละข้อประกอบด้วย:
- คำศัพท์ 1 คำ
- ตัวเลือก 5 ตัวเลือก (1 ถูก + 4 ผิด)
- ระบบสุ่มลำดับตัวเลือก
- กรอกคำตอบ 1–5 เท่านั้น

---

# 5. หน้าสรุปผล (Quiz Summary)
หลังตอบครบ ระบบจะแสดง:
- จำนวนข้อทั้งหมด
- จำนวนที่ตอบถูก
- จำนวนที่ตอบผิด
- คะแนนเป็นเปอร์เซ็นต์
- รายการคำที่ตอบผิดและเฉลย

---

# 6. รูปแบบไฟล์ vocab.csv

word,correct_choice,choice2,choice3,choice4,choice5

ตัวอย่าง:

ancient,โบราณ,ใหม่,สะอาด,เร็ว,สด  
rapid,เร็ว,ช้า,เก่า,หนา,เบา  

---

# 7. ปัญหาที่พบบ่อย

| ปัญหา | สาเหตุ | วิธีแก้ |
|-------|---------|---------|
| ไม่พบไฟล์ | ชื่อไฟล์ผิด | ตรวจสอบชื่อไฟล์ |
| ภาษาไทยเพี้ยน | ไม่ใช่ UTF-8 | เซฟไฟล์ใหม่เป็น UTF-8 |
| Start quiz ไม่ได้ | ยังไม่โหลดไฟล์ | ใช้เมนู 1 ก่อน |
| ตัวเลือกไม่สุ่ม | คำศัพท์น้อย | เพิ่มคำศัพท์ |
