# คู่มือผู้ใช้งานระบบ Vocabulary Quiz Generator

ระบบ Vocabulary Quiz Generator เป็นโปรแกรมแบบ Console ที่ใช้สำหรับฝึกคำศัพท์ภาษาอังกฤษ  
โดยอ่านข้อมูลจากไฟล์ CSV และแสดงคำถามแบบตัวเลือก **5 ตัวเลือก** ให้ผู้ใช้ตอบ โดยระบบจะสุ่มตำแหน่งคำตอบที่ถูกทุกครั้งที่เล่น

เอกสารฉบับนี้อธิบายวิธีใช้งานระบบทั้งหมด ตั้งแต่เริ่มต้นจนจบการทำแบบทดสอบ

---

# 1. ความต้องการระบบ (System Requirements)

- ระบบปฏิบัติการที่รองรับ: macOS, Windows, Linux  
- Python เวอร์ชัน: **3.11 หรือสูงกว่า**  
- ต้องมีไฟล์ `vocab.csv` อยู่ในโฟลเดอร์เดียวกับ `main.py`  
- โปรแกรมใช้งานผ่าน Terminal / Command Prompt

---

# 2. การเริ่มต้นใช้งานโปรแกรม

เปิด Terminal และเข้าสู่โฟลเดอร์โปรเจกต์:

```bash
cd 04_Implementation/src

จากนั้นรันโปรแกรมด้วยคำสั่ง:

python3 main.py

เมื่อรันสำเร็จจะพบเมนูหลักของโปรแกรม
