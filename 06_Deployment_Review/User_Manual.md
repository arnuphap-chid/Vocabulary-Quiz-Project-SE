# คู่มือผู้ใช้งานระบบ Vocabulary Quiz Generator

ระบบ Vocabulary Quiz Generator เป็นโปรแกรมแบบ Console สำหรับฝึกคำศัพท์ภาษาอังกฤษ  
โดยอ่านคำศัพท์จากไฟล์ CSV และแสดงคำถามแบบตัวเลือก 5 ตัวเลือก พร้อมสุ่มลำดับใหม่ทุกครั้ง  
ผู้ใช้สามารถตอบคำถามและดูผลสรุปได้ทันทีเมื่อทำแบบทดสอบครบ

---

# 1. ความต้องการระบบ (System Requirements)

## 1.1 ระบบปฏิบัติการที่รองรับ
- macOS  
- Windows 10 / 11  
- Linux (Ubuntu / Fedora)

## 1.2 โปรแกรมที่ต้องติดตั้ง
- Python 3.11 หรือสูงกว่า

ตรวจสอบเวอร์ชัน Python:
python3 --version

## 1.3 ไฟล์ที่จำเป็น
ไฟล์ต่อไปนี้ต้องอยู่ในโฟลเดอร์เดียวกัน:
- main.py  
- vocab.csv  

โครงสร้างไดเรกทอรี:
04_Implementation/
└── src/

    ├── main.py
    └── vocab.csv

---

# 2. วิธีเริ่มใช้งาน (How to Run)

## 2.1 เปิด Terminal หรือ Command Prompt

### macOS:
กด CMD + Space แล้วพิมพ์ “Terminal”

### Windows:
เปิด “Command Prompt” หรือ “PowerShell”

## 2.2 ไปยังโฟลเดอร์ src
cd 04_Implementation/src

## 2.3 รันโปรแกรม
python3 main.py

เมื่อโปรแกรมรันสำเร็จจะพบเมนูหลักของระบบ

---

# 3. เมนูหลัก (Main Menu)

เมนูจะแสดงดังนี้:

1. Load vocabulary file  
2. Start quiz  
3. Exit  

รายละเอียดของแต่ละเมนูอยู่ด้านล่าง

---

## 3.1 เมนูที่ 1: Load vocabulary file

ใช้สำหรับโหลดคำศัพท์จากไฟล์ CSV

### วิธีใช้งาน:
- พิมพ์ 1 แล้วกด Enter  
- หากต้องการใช้ไฟล์ vocab.csv → กด Enter อีกครั้ง  
- หากต้องการใช้ไฟล์อื่น → พิมพ์ชื่อไฟล์ เช่น words.csv  

### ผลลัพธ์:
- หากโหลดสำเร็จ → ระบบแจ้งจำนวนคำศัพท์ที่โหลดได้  
- หากไม่สำเร็จ → แจ้ง Error: File not found

## หมายเหตุ:
ผู้ใช้ต้องโหลดไฟล์ก่อนจึงจะสามารถเริ่มทำแบบทดสอบได้

---

## 3.2 เมนูที่ 2: Start quiz

ใช้เริ่มทำแบบทดสอบ

ระบบจะแสดง:

- คำถามในรูปแบบ “Question X/Y”
- คำศัพท์ภาษาอังกฤษ
- ตัวเลือก 5 ตัวเลือก (สุ่มตำแหน่ง)
- ช่องให้กรอกคำตอบ 1–5

### ตรวจสอบคำตอบ:

- ตอบถูก → Correct!  
- ตอบผิด → Incorrect. และเก็บไว้ในรายชื่อคำศัพท์ที่ผิด  
- กรอกตัวอักษร → แจ้งเตือน  
- กรอกนอกช่วง 1–5 → แจ้งเตือน  

---

## 3.3 เมนูที่ 3: Exit
ใช้เพื่อออกจากโปรแกรม

---

# 4. รายละเอียดโหมดแบบทดสอบ (Quiz Mode – Step-by-step)

ระบบจะทำงานตามลำดับดังนี้:

1. สุ่มลำดับคำศัพท์จากไฟล์ CSV  
2. สำหรับแต่ละคำ ระบบจะ:
   - เตรียมตัวเลือก 5 ช้อยส์ (1 ถูก + 4 ผิด)
   - สุ่มลำดับตัวเลือกใหม่ทั้งหมด
3. แสดงคำถามให้ผู้ใช้ตอบ  
4. ตรวจสอบคำตอบทันที  
5. บันทึกคำตอบผิด  
6. ทำซ้ำจนหมดคำถาม  

ระบบป้องกันข้อผิดพลาด เช่น:
- พิมพ์ตัวอักษร
- พิมพ์นอกช่วง
- พิมพ์ค่าว่าง

---

# 5. หน้าสรุปผล (Quiz Summary)

เมื่อทำครบทุกข้อ ระบบแสดง:

- จำนวนคำถามทั้งหมด  
- จำนวนคำตอบที่ถูก  
- จำนวนคำตอบที่ผิด  
- คะแนน (%)  
- รายการคำศัพท์ที่ตอบผิด + เฉลย  

ตัวอย่าง:
Incorrect Answers:
- ancient → โบราณ
- wise → ฉลาด

---

# 6. รูปแบบไฟล์ CSV ที่รองรับ

ไฟล์ vocab.csv ต้องเป็นรูปแบบ:

word,correct_choice,choice2,choice3,choice4,choice5

ตัวอย่างไฟล์:

ancient,โบราณ,ใหม่,สะอาด,เร็ว,สด  
rapid,เร็ว,ช้า,เก่า,หนา,เบา  

คำอธิบายคอลัมน์:
- column 1 → คำศัพท์ภาษาอังกฤษ  
- column 2 → ตัวเลือกที่ถูกต้อง  
- column 3–6 → ตัวเลือกผิด  

ข้อควรรู้:
- ต้องเป็น UTF-8  
- ต้องมี 6 คอลัมน์เสมอ  
- ห้ามเว้นว่าง

---

# 7. ปัญหาที่พบบ่อยและวิธีแก้ (FAQ)

| ปัญหา | สาเหตุ | วิธีแก้ |
|------|---------|---------|
| File not found | ชื่อไฟล์ผิด / ไฟล์ไม่อยู่ใน src | ตรวจสอบชื่อไฟล์ |
| ภาษาไทยเพี้ยน | ไฟล์ไม่ใช่ UTF-8 | เซฟเป็น UTF-8 |
| Start quiz ไม่ได้ | ยังไม่โหลดไฟล์ | ใช้เมนู 1 ก่อน |
| ตัวเลือกไม่สุ่ม | คำศัพท์น้อยเกินไป | เพิ่มคำศัพท์ |
