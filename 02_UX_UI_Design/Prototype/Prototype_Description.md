# รายละเอียด Prototype – ระบบ Vocabulary Quiz Generator (Console Application)

เอกสารฉบับนี้เป็นการอธิบาย Prototype แบบ Low-Fidelity สำหรับระบบ “Vocabulary Quiz Generator”  
แม้ระบบนี้เป็น Console Application แต่ Prototype จะเน้นแสดง **โครงสร้างหน้าจอ (Screen Layout)**  
และ **ลำดับขั้นตอนการใช้งาน (User Flow)** ให้ชัดเจน

Prototype นี้ประกอบด้วยหน้าจอหลัก 4 ส่วนคือ:

1. หน้าจอเมนูหลัก (Main Menu Screen)  
2. หน้าจอโหลดไฟล์คำศัพท์ (Load Vocabulary File Screen)  
3. หน้าจอทำแบบทดสอบ (Quiz Question Screen)  
4. หน้าจอสรุปผลคะแนน (Quiz Summary Screen)

---

## 1. หน้าจอเมนูหลัก (Main Menu Screen)

### วัตถุประสงค์
- ให้ผู้ใช้เลือกสิ่งที่ต้องการทำ เช่น โหลดไฟล์คำศัพท์, เริ่มทำแบบทดสอบ, ออกจากโปรแกรม

### ส่วนประกอบ
- ชื่อโปรแกรม
- คำอธิบายสั้น ๆ
- ตัวเลือกเมนู (1–3)
- แสดงสถานะไฟล์ที่โหลด (ถ้ามี)

### ตัวอย่าง Wireframe (แบบ Console)

========================================

Vocabulary Quiz Generator

โปรแกรมแบบฝึกคำศัพท์ด้วยระบบ Quiz
โดยอ่านข้อมูลจากไฟล์คำศัพท์ (CSV)

สถานะไฟล์ : vocab.csv
จำนวนคำถามที่โหลดได้ : 10 ข้อ

1. โหลดไฟล์คำศัพท์

2. เริ่มทำแบบทดสอบ

3. ออกจากโปรแกรม

เลือกเมนู (1-3): [ ]

## 2. หน้าจอโหลดไฟล์คำศัพท์ (Load Vocabulary File Screen)

### วัตถุประสงค์
- ให้ผู้ใช้ระบุชื่อไฟล์คำศัพท์เพื่อโหลดเข้าระบบ
- แสดงข้อความ error หรือข้อความโหลดสำเร็จ

### ส่วนประกอบ
- ช่องสำหรับใส่ชื่อไฟล์
- ปุ่มหรือข้อความ “Load file” (ในเชิงการออกแบบ)
- โซนแสดงข้อความสถานะ (สำเร็จ/ผิดพลาด)
- ปุ่มย้อนกลับเมนูหลัก

### ตัวอย่าง Wireframe

========================================

Load Vocabulary File

กรุณาระบุชื่อไฟล์คำศัพท์
(ค่าดีฟอลต์ : vocab.csv)

ชื่อไฟล์: [ vocab.csv ]

[ โหลดไฟล์ ]

สถานะการโหลด:
- โหลดคำศัพท์สำเร็จ 10 ข้อ

หรือ
- ERROR: ไม่พบไฟล์ กรุณาตรวจสอบชื่อไฟล์

[ กลับไปเมนูหลัก ]

---

## 3. หน้าจอทำแบบทดสอบ (Quiz Question Screen)

### วัตถุประสงค์
- แสดงคำถามและตัวเลือก 4 ข้อ
- ให้ผู้ใช้เลือกคำตอบ
- แสดงผลถูก/ผิดในแต่ละข้อ

### ส่วนประกอบ
- ลำดับข้อคำถาม เช่น “ข้อ 3 จาก 10”
- คำศัพท์คำถาม
- ตัวเลือก 1–4
- ช่องกรอกคำตอบ
- feedback หลังตอบ (ถูก/ผิด)

### ตัวอย่าง Wireframe

========================================

Vocabulary Quiz

ข้อที่ 3 / 10

ความหมายของคำว่า: "ancient" คืออะไร?

1. โบราณ

2. เร็ว

3. หนัก

4. สะอาด

คำตอบของคุณ (1-4): [ ]

ผลลัพธ์:
✅ ตอบถูก! หรือ ❌ ตอบผิด

[ กด Enter เพื่อไปข้อต่อไป ]
